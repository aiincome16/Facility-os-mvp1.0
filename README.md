# Facility-OS

Digitales Facility-Management-System für Reinigungsunternehmen.

## Funktionen

- QR-Code Check-In
- GPS-Prüfung
- Schichtverwaltung
- Objekt-Wiki
- Raumpläne
- Materialmanagement
- Krank- und Urlaubsverwaltung
- Vertretungs-KI
- Kundenportal
- Schlüsselbuch
- Qualitätskontrolle
- KI-Warnsystem
- Auswertungen
- Zusatzleistungen
- Abrechnung

## Architektur

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- JavaScript-Module in src/backend

### Datenhaltung (aktueller Stand)
- Demo-Daten in JavaScript
- vorbereitete Google-Sheets-Struktur

### Geplante Live-Anbindung
- Google Apps Script
- Google Sheets
- Google Drive für Bilder, Videos und PDFs

### Langfristige Ziele
- Mehrmandantenfähigkeit
- White-Label-Unterstützung
- KI-Module
- PWA und App-Version
