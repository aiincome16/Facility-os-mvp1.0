# Facility-OS

Digitales Facility-Management-System für Reinigungsunternehmen.

## Funktionen

- QR-Code Check-In
- GPS-Prüfung
- Schichtverwaltung
- Objekt-Wiki
- Raumpläne
- Materialmanagement
- Krank- und Urlaubsverwaltung
- Vertretungs-KI
- Kundenportal
- Schlüsselbuch
- Qualitätskontrolle
- KI-Warnsystem
- Auswertungen
- Zusatzleistungen
- Abrechnung

## Architektur

Frontend:
- HTML
- CSS
- JavaScript

Backend:
- Google Apps Script

Datenbank:
- Google Sheets
