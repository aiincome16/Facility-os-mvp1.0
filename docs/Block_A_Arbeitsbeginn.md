⸻

Manueller Code

Falls Kamera oder QR-Code ausfallen:

* Code manuell eingeben
* Dokumentation der manuellen Eingabe
* Auffälligkeiten markieren

⸻

Pflichtmeldungen

Wichtige Meldungen erscheinen direkt nach Arbeitsbeginn.

Darstellung:

* roter Rahmen
* hohe Priorität

Meldung kann:

* bestätigt
* erledigt

werden.

Erledigung wird gespeichert.

⸻

Tagessteuerung

Enthält:

* Objekt
* Arbeitszeit
* Mitteilungen
* Kundenwünsche
* Material
* offene Punkte
* Arbeitsende

⸻

Ziel

Mitarbeiter sollen möglichst den gesamten Tag aus der Tagessteuerung heraus arbeiten können.
